# Investment-Accounts-Management-Islamic-Investment-Banking
Built an efficient-frontier portfolio optimization model in Python/SQL for risk-return aligned asset allocation under userdefined constraints (risk density, credit RWA ratio).– Designed k-mode clustering and PCA-based visualization pipelines with python batch processing for large-scale investment data under SLA constraints.– Developed forecasting and fund simulation models using SARIMAX, XGBoost Regressor, and CatBoost Regressor; exposed outputs via REST APIs for UI integration.– Enabled automated tagging of accounts worth MYR 56B, contributing to MYR 4B RWA savings.
